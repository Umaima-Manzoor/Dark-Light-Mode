Seamless Dark & Light Modes

Click the toggle in the navbar to switch themes. Your choice is automatically saved to Local Storage and will persist even after you close or refresh the browser.

How It Works

Built with pure CSS variables and vanilla JavaScript for maximum performance.

Light Mode

Clean, bright interfaces optimized for daytime use and well-lit environments.

Dark Mode

Reduces eye strain in low-light conditions while saving battery on OLED screens.

Local Storage

Your preference is saved directly in your browser, requiring no backend database.

System Detection

Automatically matches your operating system's theme if you haven't made a manual choice yet.

Pixel-Perfect Transitions

Notice how every element on this page—backgrounds, text, cards, borders, and shadows—transitions smoothly without flashing or jumping. This is achieved using CSS custom properties (variables) mapped directly to the root element.

// Check Local Storage on load
const savedTheme = localStorage.getItem('theme');

if (savedTheme) {
  document.documentElement.setAttribute(
    'data-theme', savedTheme
  );
} else if (window.matchMedia(
  '(prefers-color-scheme: dark)'
).matches) {
  setTheme('dark');
}
